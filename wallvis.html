Swap Market Exchang Rate and Wall Vis

An interactive wall exhibition. Users can input their value system using a controller which then drives the wall visualisation.The project runs on a Node Red server on a Raspberry Pi, driving ESP8322 Arduino modules through MQTT. Group project with Ailie Rutherford and University of Edinburgh, I was responsible for programming and concept design.

I was also part of the initial testing phases of the wall visualisation proejct for Design Informatics at the University of Edinburgh. I was responsible for the module concept design and the module programming. These modules were then workshopped with industry partners.
